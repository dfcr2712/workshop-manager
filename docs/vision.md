# Vision

## Workshop Manager

Workshop Manager é uma aplicação de gestão destinada a pequenas oficinas automóveis e de motociclos.

O objetivo é centralizar toda a informação relacionada com clientes, veiculos, ordens de serviço e histório de manutenção numa única plataforma, substituindo processos manuais ou folhas de cálculo.

Ao longo do desenvolvimento, este projeto irá evoluir para acompanhar as tecnologias aprendidas durante a licenciatura, tornando-se um projeto de portfólio completo.
